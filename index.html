PCU Enrollment Form

This is a simple HTML-based enrollment form for students to enroll themselves in courses. The form records the most critical student information; these include the name of the student, age, contact details, and chosen courses. It ensures the easy and organized way by which users submit their data with structured input fields, dropdowns, and checkboxes. It could be a starter project in web development, involving the actual application of some HTML tags and the main concepts in handling forms.

The Multiplication Table

The Multiplication Table HTML project for the first five digits is a very simple web-based application, showing results of multiplication from 1 up to 5. Right after opening the page, users will be able to see the table of multiplication of the number input (x) by 1, 2, 3, 4, and 5 in an orderly way. This project clearly illustrates the explanation of basic HTML table structures with practice of CSS for styles. It is an excellent starting point for both JavaScript and the simple static display of data, especially with loops and formatting in development.

USED STACK:

  • html5
  • css4

Résumé

The Résumé HTML project is a web application which shows a professional résumé in a clean, visually appealing way. The sections are usually: contact information, education, skills, work experience, and competencies. This structure makes it easy to read because of its layout. This project will demonstrate how HTML can be used to structure content, CSS to style, and JavaScript for adding interactive elements. This is a fantastic exercise to get a feel for responsive design, layout techniques, and content organization so will be a great practical application of using HTML and CSS to make a working digital resume.

USED STACK:

  • html5
  • css4

Responsive Webpage

The Birthmonth Calendar Responsive Webpage project is showing a calendar layout of the users' birth month as a responsive web application that adapts to the screen size. It uses HTML for the structure and CSS, including Flexbox and Grid, for styling and laying out the application without any issues on either desktop or mobiles with readability and ease of use. It is a good project to practice on date-based layouts, visual styling, and responsive design while giving the user a very intuitive calendar experience.

USED STACK:

  • html5
  • css4
